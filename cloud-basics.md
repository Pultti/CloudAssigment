# Basics of Cloud Computing

## Types of Cloud Services
- IaaS (Infrastructure as a Service)
- PaaS (Platform as a Service)
- SaaS (Software as a Service)

## Deployment Models
- Public Cloud
- Private Cloud
- Hybrid Cloud

## Advantages
- Cost-Efficiency
- Scalability
- Flexibility

## Disadvantages
- Security Concerns
- Limited Control
- Vendor lock-in
